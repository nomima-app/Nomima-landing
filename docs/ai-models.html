AI & integrations

AI models

Nomima doesn’t lock you into anyone’s AI. Bring your own key for Claude, GPT or Gemini. Summon connects only to current frontier models, so you always get each provider’s strongest reasoning.

Bring your own key. AI in Nomima (the Summon assistant) is free — you just supply an API key from one of the supported providers. Everything else in Nomima — including the Knowledge Graph, Smart Views and search — works without any AI at all.

Frontier models only. Summon connects exclusively to each provider’s current frontier tier so you always get the strongest reasoning, tool-use and writing. Older and self-hosted/local models are no longer offered.

Setting up a provider

  1. Open Settings → AI Models.
  2. Pick a provider card and paste your API key.
  3. Choose a model — Nomima lists the current frontier models available for that provider.
  4. Click Test — Nomima makes one small call to confirm the credentials work.
  5. Set your default provider; that’s what Summon uses unless told otherwise.

How your keys are protected

  • Keys are stored in the macOS Keychain, not in a config file. In the UI you’ll only ever see the last few characters.
  • Keys are used for exactly one thing: calling the provider you configured, when you invoke AI. Nomima has no server of its own and never proxies your traffic.
  • Delete a key any time from the same panel.

If you never configure a provider, Nomima simply never makes an AI call. The app is fully functional as a private, offline notebook.

If something fails

Nomima tells you exactly what went wrong rather than failing silently: an invalid key (fix the key), quota exhausted (top up with your provider), rate limited (wait a moment and retry), or a network error (check your connection).
