Publishing

Put a notebook on the web as a read-only page that looks exactly like it does in your editor — protected by a view-code, and kept up to date automatically as you keep writing.

Pro feature. Publishing requires a Nomima Pro license. Everything else about your vault stays local — only the notes you explicitly publish ever leave your Mac.

Publishing a note

  1. Open the notebook and click its publish badge in the note header.
  2. Nomima gives you two things: a private link at read.nomima.io and a view-code.
  3. Share them however you like — send the link and code separately for extra care, or share the single combined link that fills the code in automatically.

Anyone opening the bare link is asked for the code first; without it, the page shows nothing.

What readers see

  • A pixel-faithful replica of your note — the published page uses the editor's own rendering and your theme, so every callout, Smart-Tag pill, kanban board, table, equation and diagram looks exactly as it does in the app.
  • It's read-only — readers can't edit, and none of your vault's structure travels with it. The raw note data is never published, only the rendered page.
  • The page shows when the note was created and last edited, so readers know how fresh it is.

It stays up to date by itself

  • Edits auto-update the page. Keep writing — changes flow to the published version automatically (briefly debounced, and skipped entirely when nothing actually changed).
  • Offline-friendly. Publish, update or unpublish without a connection and the action waits in a durable queue, sending when you're back online — even across an app restart.
  • The badge tells the truth. The note header shows the live state at all times: Draft → Publishing → Published, plus Update queued or Failed when something is pending or wrong.

Sharing attached files

Files attached to the notebook are not shared by default. The publish menu lists them with checkboxes — tick the ones you want readers to have and they appear as download chips at the top of the published page. Everything unticked stays on your Mac.

Privacy & control

  • Only people with both the link and the view-code can read the page.
  • Unpublish any time — the page is taken down, including from the offline queue if you're not connected.
  • Publishing is the only Nomima feature that uploads note content anywhere, and it does so only for the notes you choose — see the complete privacy story.
