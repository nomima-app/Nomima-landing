Paste, templates & shortcuts

The quality-of-life layer: paste that understands where text came from, reusable note templates, comments, and a keyboard that always does what you expect.

SmartCopy — paste that reads the room

Paste with ⌘V and Nomima looks at what you’re pasting and where it came from, then converts it into real blocks instead of a wall of flat text:

  • AI chat answers (Claude, ChatGPT…) unpack into proper headings, lists, checkboxes, Databases, code blocks with the right language, Mermaid diagrams and rendered math.
  • Source-aware cleanup — Slack pastes lose their markup junk but keep links; Outlook bullet salads become real lists; Excel ranges and web tables become Databases; GitHub code keeps its language.
  • Tags on arrival — @names, #tags and dates in pasted text become smart pills. Code is exempt: a # in a comment stays a comment.
  • Links, files and images too — pasted hyperlinks become link pills, files on the clipboard import straight into your vault, and remote images are pulled local so notes stay offline-readable.
  • Safe by construction — scripts and javascript: links are stripped; pasting can never run code.

Want it raw? ⌘⇧V pastes plain text with no interpretation at all.

Templates

  • Use a template (free): when creating a notebook, pick from 20 built-ins across five categories (Work, Engineering, Research, Personal, Creative) — every seeded date lands relative to today, so a sprint board is live the moment it opens.
  • Save as template: open the ⋯ menu on any notebook and choose Save as template… — the note's block structure is captured as a reusable template. Templates are listed newest-first in the picker.
  • Rename a template: select any user-saved template in the picker to rename it.
  • Delete a template: remove any user-saved template permanently from the picker.

Saving a template captures the note's block structure at that moment. Later edits to the original notebook do not update the saved template.

Selection toolbar & comments

Select any text and a floating toolbar appears above the selection. Its buttons are grouped left to right:

SectionWhat it does
Inline formatBold, Italic, Underline, Strikethrough, Inline code — toggle any combination on the selected text.
Smart tagsConvert the selection to a #hashtag or @mention pill. Only shown for selections of 1–3 words — longer phrases are not offered because they cannot form a single tag.
Text colour8 preset colours (red, orange, yellow, green, teal, blue, violet, pink) plus a reset-to-default option.
Highlight6 translucent highlight colours (yellow, green, sky, purple, pink, orange) plus a clear option.
AlignmentLeft, Centre, Right — applies to the paragraph or heading that contains the selection.
LinkOpens an inline URL input; Enter applies the link, Esc cancels. Click the button again on an already-linked selection to remove the link.
CommentOpens a new comment thread anchored to the selected text (see below).

The Smart tag buttons only appear when your selection is 1–3 words long. For longer selections they are automatically hidden.

Comment threads

Clicking the comment button creates a thread anchored to the selected text. The quoted excerpt is stored with the thread so you always know what it refers to. Threads work like this:

  • Replies — anyone can add further entries to any thread at any time.
  • Resolve — mark a thread done; it fades out of the way but stays in the note's history.
  • Reopen — a resolved thread can be reopened if the conversation needs to continue.
  • Delete — remove a thread and all its entries permanently.

Every open comment thread appears as an accent-coloured dash on the Smart Scrollbar. Hover a dash to peek at the thread; click it to jump straight to the anchored text. Resolved threads do not show a dash.

Export & sharing

Any notebook exports as a single continuous, exact-colour PDF from its ⋯ menu. Two paths are available:

  • Download — presents a save-file dialog and writes the PDF to a path you choose on disk.
  • Share — captures the PDF to a temporary file and immediately presents the macOS Share sheet anchored to the share button, so you can send it via Mail, Messages, AirDrop, Slack, Teams, or any other sharing service installed on your Mac.

The export uses the native macOS WKWebView PDF API and requires macOS 11 or later. The resulting PDF captures the notebook exactly as it appears on screen, including all block styles and colours. PDF export is a core free feature.

PDF export is only available on macOS.

Staying up to date

Nomima checks for new versions quietly (it never auto-installs anything). When an update exists you’ll see a dismissible banner, and Settings → About always has a Check for updates button with a download link to the latest release.
