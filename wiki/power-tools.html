Paste, templates & shortcuts

The quality-of-life layer: paste that understands where text came from, reusable note templates, comments, and a keyboard that always does what you expect.

SmartCopy — paste that reads the room

Paste with ⌘V and Nomima looks at what you’re pasting and where it came from, then converts it into real blocks instead of a wall of flat text:

  • AI chat answers (Claude, ChatGPT…) unpack into proper headings, lists, checkboxes, tables, code blocks with the right language, Mermaid diagrams and rendered math.
  • Source-aware cleanup — Slack pastes lose their markup junk but keep links; Outlook bullet salads become real lists; Excel ranges become tables; GitHub code keeps its language.
  • Tags on arrival — @names, #tags and dates in pasted text become smart pills. Code is exempt: a # in a comment stays a comment.
  • Links, files and images too — pasted hyperlinks become link pills, files on the clipboard import straight into your vault, and remote images are pulled local so notes stay offline-readable.
  • Safe by construction — scripts and javascript: links are stripped; pasting can never run code.

Want it raw? ⌘⇧V pastes plain text with no interpretation at all.

Templates

  • Use a template (free): when creating a notebook, pick from 20 built-ins across five categories (Work, Engineering, Research, Personal, Creative) — every seeded date lands relative to today, so a sprint board is live the moment it opens.
  • Save as template Pro: open the ⋯ menu on any notebook and choose Save as template… — its block structure becomes reusable. Rename or delete templates from the same picker.

Selection toolbar & comments

Select any text and a floating toolbar appears — formatting, multi-color highlights, turning the selection into a Smart Tag, and adding a comment. Threads support replies and a resolved state. Comment locations show as accent-colored dashes on the Smart Scrollbar — hover to peek, click to jump to the thread. Resolved threads fade out of the way.

Export & sharing

Any notebook exports as a single continuous, exact-color PDF from its ⋯ menu — Download saves it, Share hands it straight to the macOS share sheet (Mail, AirDrop, Messages, Slack…). A text-size toggle lives in the same menu, and any notebook can switch to a wide view when you need the room. All free, like the rest of core Nomima.

Staying up to date

Nomima checks for new versions quietly (it never auto-installs anything). When an update exists you’ll see a dismissible banner, and Settings → About always has a Check for updates button with a download link to the latest release.
