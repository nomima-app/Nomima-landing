Smart Tags

Where typing
becomes knowledge

As you write, certain words become Smart Tags — inline chips the whole app can reason about. No @ or # required. A ghost suggestion appears as you type a recognized name — press Tab to confirm.

  • Date tags count down live. Write "in 3 hours" and a timer appears. When it passes, the tag turns red and shows "done? · reschedule?" right on the chip.
  • #complete is universal. Apply it to any block — task, Kanban card, Gantt row, entire section — and it strikes through across every view, graph, and search at once.
  • Hover any tag to see every block where that person or project appears, across all your notebooks.
  • Custom types. Add your own mention and hashtag categories — label, icon and color — in Settings. They appear in Smart Tags, the graph legend, and views.

Knowledge Graph

See the shape
of your thinking

Every note you write quietly builds a graph. People, projects, ideas, tasks and dates become nodes; the connections become edges. It refreshes live as you write — no "rebuild" button.

  • Note cover graph. Every note opens above its own live 3D graph — a hub, entities it mentions, and a ring of related notes. Grab and spin it.
  • Full vault graph. Full-screen 3D WebGL — glowing color-coded orbs, animated synapses. Cluster by Notebook, Theme, Person, Date or Type.
  • Lenses & filters. Filter by entity kind, scrub a time-range slider, search to focus a node, click any node for its full relationship panel.
  • Smart scrollbar. Every note has a minimap rail — colored marks for people, hashtags, overdue dates (red), links. Hover to magnify, click to jump.

Home

Your daily agenda, three ways

Home answers "what should I focus on today?" — three different lenses over your entire vault. Summary chips keep a live count of overdue, today, upcoming and running reminders in every mode.

By Time

A structured agenda bucketed by urgency — Overdue (red), Today, Next Few Days (with friendly labels like "tomorrow" and "in 3 days"), and Later. Recently Completed surfaces when nothing is pending.

By @Mention

Everything grouped by the person or entity it mentions, overdue first. Tap any name to open a focused view for just that person — the pill highlighted in every row.

By #Hashtag

The same grouped view, organized by topic or project tag. Tap a hashtag to see everything tagged with it, bucketed by urgency — your project tracker in one tap.

Reminders

Set it where
you think it

Reminder blocks are inline countdowns. Type "in 3 hours" or "by end of day" and press Enter to start the clock. Collapses to a tidy row and counts down live.

  • macOS notifications with Clear and Snooze quick actions. Tap to jump to that block — even from a cold start.
  • Overdue badges everywhere — sidebar, each notebook, and the macOS Dock icon. All three always agree.
  • Snooze from anywhere — block, Home, Notifications. 30 min, 1h, 3h, or 1 day.
  • Any dated item fires. Any block with a passed date notifies once, deduplicated across launches.

Summon — your AI sidekick

The bottom-right wand, ⌘⇧Space, or /summon opens a docked assistant whose conversation persists as you move around. It searches notes, reads notebooks, checks reminders — grounded in your vault.

It does real work

Summon builds genuine Nomima blocks — checklists, callouts, tables, kanban — creates whole notebooks, and proposes templates you Accept or Reject inline. Any answer pins into a note as a Section.

Web, files & shell — gated

Optional power tools in Settings → Summon Agent: keyless web search, file access in Documents / Desktop / Downloads, and an allowlisted shell. Writes and commands pause for an inline Allow / Deny.

Publishing — Pro

Put a note on the web.
Exactly as you see it.

One click publishes a notebook to read.nomima.io as a read-only page that is a pixel-faithful replica of your editor — every callout, pill, kanban and equation, in your theme.

Private by default

Every published note is gated by a view-code. Share the link and the code separately, or a single link with the code built in. The raw note data is never published.

Always up to date

Keep editing — the published page updates automatically. Publishing while offline queues durably and sends when you reconnect; the note's badge shows the live state.

Share files, if you choose

Files attached to the notebook stay private unless you tick them — selected ones appear as download chips at the top of the page. Unpublish any time.
